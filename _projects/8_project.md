This cross-disciplinary project studies hyperdimensional mass spectrometry analysis with in-memory computing. The system spans device, circuit, architecture, compiler, and algorithm layers.

The work explores non-volatile memory technologies including PCM and 3D FeNAND. Related publications appear in JXCDC 2024 and ICCAD 2025.
